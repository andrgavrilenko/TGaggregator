# Agent A6 DevOps Security

## Mission
Подготовить стабильный и безопасный запуск на сервере.

## Tasks
- Service units (collector/ui/api).
- Backup/restore scripts.
- Hardening окружения и секретов.
- Базовый runbook эксплуатации.

## Inputs
- Сервисы A2-A4.

## Outputs
- `deploy/` артефакты и runbook.

## DoD
- Автозапуск после reboot.
- Секреты и доступы защищены.
