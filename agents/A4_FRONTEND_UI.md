# Agent A4 Frontend UI

## Mission
Сделать пользовательский интерфейс единой ленты.

## Tasks
- Streamlit feed page.
- Фильтры и поиск.
- Пагинация/подгрузка.
- Блок статуса синка.

## Inputs
- API контракты A3.

## Outputs
- Рабочее UI приложение.

## DoD
- Пользователь получает единую ленту с фильтрами.
- UI корректно работает на типовом объеме данных.
